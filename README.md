# Ауыл Mafia — Түнгі Құпия

Қазақ ауылы стиліндегі браузерлік party-ойын. Классикалық «Мафия / Werewolf»
негізінде жасалған: ойыншылар сілтеме арқылы бөлмеге қосылады, әркім құпия рөл
алады, түнде рөлдер әрекет жасайды, күндіз ауыл талқылап дауыс береді.

**Ауыл** барлық Хуторлықтарды әшкерелесе жеңеді. **Хуторлықтар** ауылмен
теңессе жеңеді. Қатыгездік, қан, зорлық жоқ — тек ойынша «ойыннан шығу».

- **Атауы:** Ауыл Mafia
- **Субтитр:** Түнгі Құпия
- **Тіл:** қазақша (UI толығымен қазақ тілінде)
- **Ойыншылар:** кемінде 4 · ұсынылады 5–8 · максимум 10

> Ескерту: AI тек **әзірлеу құралы** ретінде қолданылды (код, дизайн, рөлдер,
> баланс, README, дебаг). Ойынның өзі **ешқандай AI API талап етпейді** —
> Gemini, OpenAI, Claude кілттері қажет емес.

## Қалай ойнау керек

1. Басты бетте атыңызды жазып **«Бөлме құру»** немесе кодпен **«Бөлмеге кіру»**.
2. Лоббиде шақыру сілтемесін достарға жіберіңіз. Хост **«Бот қосу»** арқылы
   демо/тест үшін бот қоса алады.
3. Ойыншы саны ≥ 4 болғанда хост **«Ойынды бастау»** батырмасын басады.
4. Әркім құпия рөлін көреді (15 сек).
5. **Түн:** рөлдер жасырын әрекет жасайды.
6. **Таң:** түнгі жаңалық жарияланады.
7. **Талқылау:** чат арқылы күдіктілерді талқылайсыз (90 сек).
8. **Дауыс беру:** бір ойыншыға дауыс (30 сек).
9. Жеңіс шарты орындалғанша қайталанады, соңында барлық рөл ашылады.

### Фазалар мен таймерлер
| Фаза | Уақыт |
|------|-------|
| Рөлмен танысу | 15 сек |
| Түн | 45 сек (барлық әрекет түскенде ертерек) |
| Таңғы жаңалық | қысқа |
| Талқылау | 90 сек |
| Дауыс беру | 30 сек |
| Күн қорытындысы | 10 сек |

## Рөлдер

| Рөл | Команда | Түнгі әрекет |
|-----|---------|--------------|
| **Ауыл тұрғыны** | Ауыл | — (тек дауыс) |
| **Учаскелік** | Ауыл | Бір ойыншыны тексереді: «таза» / «күдікті» |
| **Емші** | Ауыл | Бір ойыншыны қорғайды (қатарынан бір адамды 2 рет емес; өзін 1 рет) |
| **Таксист** | Ауыл | Екі ойыншыны ауыстырады (оларға бағытталған әрекеттер орын ауыстырады) |
| **Аким** | Ауыл | Тең дауыста шешеді; алғаш шықса бір рет аман қалады |
| **Хуторлық** | Хуторлықтар | Түнде бір ойыншыны таңдайды (көпшілік дауыспен) |
| **Бандит** | Хуторлықтар | Бір ойыншының түнгі әрекетін бұғаттайды |
| **Өсекші** | Ауыл | Әр таңда бұлыңғыр кеңес алады |
| **Хаосшы** | Бейтарап | Әрекет бұғаттайды; финалда (2 ойыншы) тірі қалса жеке жеңеді |

Хуторлықтар туралы: «Хуторлықтар түнде бір ойыншыны таңдайды. Егер оны Емші
қорғамаса, ол ойыннан шығады.»

### Рөл саны (ойыншы санына қарай)
- **4:** Учаскелік, Емші, Хуторлық, Ауыл тұрғыны
- **5:** + Хаосшы
- **6:** Учаскелік, Емші, Таксист, Хуторлық ×2, Ауыл тұрғыны
- **7:** + Өсекші
- **8:** Учаскелік, Емші, Таксист, Аким, Өсекші, Хуторлық, Бандит, Ауыл тұрғыны
- **9–10:** қосымша Ауыл тұрғыны (баланс үшін 10-да тағы бір Хуторлық)

## Рөл суреттерін қалай ауыстыруға болады

Рөл суреттері `frontend/public/roles/` папкасынан ізделеді.

PNG файлдарын **дәл осы атаулармен** қойыңыз:

```
villager.png
uchastkovyi.png
emshi.png
taxi.png
akim.png
hutor.png
bandit.png
spletni.png
haosshy.png
bot.png
```

- Ұсынылатын өлшем: **шаршы 512×512 PNG**.
- Сурет болмаса, ойын автоматты түрде **кірістірілген SVG/эмодзи fallback**
  көрсетеді — ештеңе бұзылмайды.
- Шынайы саясаткерлердің фотосын немесе авторлық құқықпен қорғалған суреттерді
  **рұқсатсыз пайдаланбаңыз**.
- Суретті ауыстырғаннан кейін қайта build/деплой жасаңыз.

## Технологиялар

- **Frontend:** React + Vite + таза CSS (ауыр UI framework жоқ)
- **Backend:** FastAPI + Python, HTTP polling (1–2 сек), жадтағы бөлмелер
  (дерекқор жоқ, авторизация жоқ, WebSocket жоқ)
- **Деплой:** Docker / Hugging Face Docker Space

## Жергілікті іске қосу

```bash
pip install -r requirements.txt
cd frontend
npm install
npm run build
cd ..
uvicorn backend.main:app --host 0.0.0.0 --port 7860
```

Содан соң браузерден `http://localhost:7860` ашыңыз.

Әзірлеу кезінде Vite dev-серверін бөлек қосуға болады (`cd frontend && npm run dev`),
ол `/api` сұрауларын 7860 портына проксилейді.

Логиканы тексеру (симуляция):
```bash
python backend/sim_test.py
```

## Docker / Hugging Face деплой

Жоба түбіндегі `Dockerfile` frontend-ті build жасап, backend-пен бірге орайды.

```bash
docker build -t auyl-mafia .
docker run -p 7860:7860 auyl-mafia
```

Қосымша `PORT` айнымалысын қолдайды (әдепкі 7860), `0.0.0.0`-де тыңдайды.

**Hugging Face Docker Space:** репозиторий түбіндегі `Dockerfile`-дан build
жасалады. Space-ке осы репозиторийді жүктесеңіз жеткілікті — қосымша баптау
қажет емес.
